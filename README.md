# 星吞龍

一個手機直向的網頁遊戲原型，核心是拖曳小龍、用龍嘴吞彈、把吞下的彈反吐回敵人。玩法氣質參考吞噬彈幕類龍題材遊戲，但名稱、美術、數值與程式都為原創。

## 目前系統

- 主畫面：資源、出戰、龍幻化、神器、關卡、養成、召喚入口。
- 龍角色：已有 10 種原創龍，每隻有不同屬性、定位、配色、被動與戰鬥數值。
- 神器：每隻龍綁定 1 件專屬神器，神器頁會顯示 10 件生成圖示神器陣列；解鎖後可強化該龍的持續型專屬大絕。
- 幻化：每隻龍可切換原生、吞星、迅翼、龍息形態，形態會改變血量、速度、吞噬槽、反吐倍率與部分戰鬥效果。
- 關卡主線：第一章已有 `1-1` 到 `1-10` 共 10 個主線關卡，每關有章名、任務、開場敘事、通關推進文字與 Boss。
- 關卡：每關至少 10 波，每波約 20 秒，小怪、菁英、Boss 的流程已接上；菁英/Boss 會施放預警雷射與範圍攻擊，通關會回到主畫面並給金幣與龍晶；不同關卡有敵人血量、彈速、危險招式節奏與主題背景差異；局內技能包含元素範圍傷、緩速、吞噬強化、連鎖雷、魂能收割、符文護盾與 Boss 增傷。
- 養成：龍強化、升星、龍影召喚、六角晶體裝備、技能研究、掛機收益領取都有最小可玩版本。
- 美術：`assets/dragons-clean/dragon-*.png` 是從設定集整理出的透明底 UI 龍圖，`assets/dragons/` 保留原整理版；`assets/battle-dragons/battle-*.png` 是各龍專屬上帝視角戰鬥 CG；`assets/stage-backgrounds/` 內有溪谷、海礁、熔火、雷霜、虛空與祭壇直式戰鬥背景；`assets/artifacts/*.png` 是 10 件神器生成圖示；`assets/dragon-sheets/` 保留原設定集參考圖，`assets/visuals.svg` 保留作為關卡 sprite。
- 戰鬥表現：主畫面與龍卡使用設定集整理出的 PNG；關卡中的龍使用對應的 `battle-*.png` 俯視 CG，再由 canvas 做飛行擺尾、分層呼吸、張嘴吸收、吸入漩渦風場、子彈捲入拖尾、能量環與反吐爆發動畫。
- 存檔：單人進度會寫入 Firebase Firestore `singlePlayerSaves/{gameId__deviceId__slot}`，並同步保留一份本機快取。

## 開啟

直接用瀏覽器開啟 `index.html` 即可遊玩。若要模擬手機安裝體驗，可以用本機伺服器啟動：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然後開啟 `http://127.0.0.1:4173/`。

## 操作

- 主畫面可切換龍、幻化形態、解鎖龍、升級龍、解鎖神器、選關卡。
- 關卡分頁可查看章節主線、任務目標與通關後故事推進。
- 關卡分頁使用主線路線節點呈現 `1-1` 到 `1-10`，顯示鎖定、已通關、任務與獎勵。
- 養成分頁可用龍晶進行龍影召喚，隨機解鎖尚未取得的龍。
- 直接拖曳龍移動；按住畫面時會張嘴吸收，放開就把吞下的彈幕釋放成反擊吐息。
- 右下角圓形按鈕也保留同樣的吞吐操作，方便桌面或單手測試。
- 解鎖神器後，戰鬥中累積大絕能量；能量滿會自動啟動一段專屬持續攻擊，不需要額外按鈕。
- 測試任一龍的戰鬥 CG 可用 `?battle=1&dragon=ember`，也可搭配 `&absorb=1` 或 `&ultimate=1` 看吸收與自動大絕。
- 戰鬥 HUD 會顯示關卡進度條、目前波次、每波倒數，換波與 Boss 登場會跳出提示；10 波結束後進入 Boss 戰。
- 養成分頁的寶物裝備以六角晶體卡片排列，點擊晶體或強化按鈕可消耗金幣提高裝備等級，生命、攻擊、速度、吞噬、防禦與大絕會分別套用；掛機收益會顯示可領取金幣並可一鍵領取。
- 鍵盤也可用方向鍵或 WASD 移動，空白鍵吞吐。

- 砂遺古龍 / 砂遺秘卷：沙漠遺跡，古砂秘寶。
- 櫻風舞龍 / 櫻風花鈴：櫻花輔助，櫻風花燈。
- 烈焰椒龍 / 辣焰椒冠：辛辣爆發，辣焰飛椒。
- 星夢幻龍 / 星夢法環：星空幻術，星夢回流。
- 泡泡海龍 / 泡海珍珠：泡泡治癒，泡海漩渦。
- 森語芽龍 / 森語芽杖：森林守護，森語再生。
- 霆電迅龍 / 霆電迅核：迅捷雷擊，霆電連鎖。
- 霜晶雪龍 / 霜晶雪杖：冰霜控制，霜晶結界。
- 月影幽龍 / 月影幽刃：暗影連擊，月影逆鳴。
- 齒輪機龍 / 齒輪策盤：科技策略，齒輪震波。
