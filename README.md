# 星吞龍

一個手機直向的網頁遊戲原型，核心是拖曳小龍、用龍嘴吞彈、把吞下的彈反吐回敵人。玩法氣質參考吞噬彈幕類龍題材遊戲，但名稱、美術、數值與程式都為原創。

## 目前系統

- 主畫面：資源、出戰、下一步養成導引、關卡任務簡報、龍幻化、神器、關卡、養成、召喚入口與召喚結果揭曉。
- 龍角色：已有 10 種原創龍，每隻有不同屬性、定位、配色、被動與戰鬥數值。
- 神器：每隻龍綁定 1 件專屬神器，神器頁會顯示 10 件生成圖示神器陣列；啟動神器後才會解放該龍能量滿自動施放的持續型專屬大絕，後續強化會提升持續時間與傷害。
- 幻化：每隻龍可切換原生、吞星、迅翼、龍息形態，形態會改變血量、速度、吞噬槽、反吐倍率與部分戰鬥效果。
- 關卡主線：已有第一章 `1-1` 到 `1-10`、第二章 `2-1` 到 `2-10` 共 20 個主線關卡，每關有章名、任務、開場敘事、通關推進文字與 Boss。
- 關卡：每關至少 10 波，每波約 20 秒，小怪、菁英、Boss 的流程已接上；菁英/Boss 會施放預警雷射與範圍攻擊，Boss 會依關卡主題使用星核扇幕、潮泡迴環、熔牙重砲、霆針鎖線、裂隙回流或龍王祭印；通關或失敗結算會顯示分數、戰鬥評價、帶回資源、擊破數、吞彈、反吐、受擊、大絕、共鳴與下一步操作建議；不同關卡有敵人血量、彈速、危險招式節奏、出怪壓力、主題背景與環境變體差異；局內技能包含元素範圍傷、緩速、吞噬強化、連鎖雷、魂能收割、符文護盾與 Boss 增傷，特定兩技能會觸發火雷、冰盾、吞魂與反吐矩陣共鳴。
- 養成：龍強化、升星、龍影召喚、六角晶體裝備、技能研究、技能搭配、掛機收益領取都有最小可玩版本；搭配的戰術會在出戰時自帶 Lv.1，並提高局內升級選項出現率；局內湊齊指定技能組合時，HUD 會顯示技能共鳴並改變輸出、擊殺回能或受擊控場。
- 美術：`assets/dragons-clean/dragon-*.png` 是從設定集整理出的透明底 UI 龍圖，`assets/dragons/` 保留原整理版；`assets/battle-dragons/battle-*.png` 是各龍專屬上帝視角戰鬥 CG；`assets/stage-backgrounds/` 內有溪谷、海礁、熔火、雷霜、虛空與祭壇直式戰鬥背景；`assets/artifacts/*.png` 是 10 件神器生成圖示；`assets/dragon-sheets/` 保留原設定集參考圖，`assets/visuals.svg` 保留作為關卡 sprite。
- 標題徽章：主畫面不再顯示原本的圖片式標題 icon，改用 CSS 繪製的發光龍紋徽章。
- 畫面穩定：戰鬥外框會用實際 viewport 高度同步 canvas，降低手機或內嵌瀏覽器底部溢出與畫面重複感。
- 戰鬥表現：主畫面與龍卡使用設定集整理出的 PNG；關卡中的龍使用對應的 `battle-*.png` 俯視 CG，再由 canvas 做較收斂的飛行擺尾、分層呼吸、張嘴吸收、吸入漩渦風場、子彈捲入拖尾、能量環與反吐爆發動畫；10 種龍的普通攻擊有不同彈道、拖尾、命中狀態與反吐/大絕小彈形狀，整體輸出已下修以保留走位與吞彈壓力。
- 敵人美術：小怪、快怪、重甲怪、菁英與 Boss 會依關卡主題色繪製成魂燈、翼刃、熔甲、祭司與龍王影等主視覺 skin。
- 存檔：單人進度會寫入 Firebase Firestore `singlePlayerSaves/{gameId__anonymous_uid__slot}`；若 Anonymous Auth 尚未啟用，會暫時退回 `singlePlayerSaves/{gameId__device_deviceId__slot}`，並同步保留一份本機快取。啟用 Anonymous Auth 後會自動檢查同裝置舊的 device 備援文件，選擇本機、匿名雲端、裝置備援中進度較完整的一份寫回目前使用者文件。養成頁會顯示目前存檔來源、文件路徑與 owner key，方便在 Firebase console 對照不同使用者。

## 開啟

直接用瀏覽器開啟 `index.html` 即可遊玩。若要模擬手機安裝體驗，可以用本機伺服器啟動：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然後開啟 `http://127.0.0.1:4173/`。

## 操作

- 主畫面可切換龍、幻化形態、解鎖龍、升級龍、解鎖神器、選關卡。
- 主畫面會依目前戰力、關卡建議值、資源、神器狀態與技能槽推薦下一個養成入口。
- 出戰前會顯示關卡簡報，整合章節、劇情、任務、Boss、環境變體、獎勵、出戰龍與技能搭配，再開始戰鬥。
- 關卡分頁可查看章節主線、任務目標與通關後故事推進。
- 關卡分頁使用主線路線節點呈現 `1-1` 到 `2-10`，顯示鎖定、已通關、任務與獎勵。
- 養成分頁可用龍晶進行龍影召喚，隨機解鎖尚未取得的龍，並以結果卡顯示新龍大圖、屬性、定位與星等；全龍已取得時會轉為龍魂共鳴升星。
- 直接拖曳龍移動；按住畫面時會張嘴吸收，基礎吸收口徑需要更貼近龍嘴，放開就把吞下的彈幕釋放成反擊吐息。
- 戰鬥中不顯示吞吐或大絕操作按鈕；玩家只靠拖曳龍本身、按住吸收、放開反吐。
- 啟動神器後，戰鬥中累積大絕能量；能量滿會自動啟動一段專屬持續攻擊，不需要額外按鈕。未啟動神器時會保留一般吞噬反吐玩法並提示去神器頁解封。
- 測試任一龍的戰鬥 CG 可用 `?battle=1&dragon=ember`，也可搭配 `&absorb=1` 或 `&ultimate=1` 看吸收與自動大絕。
- 戰鬥 HUD 會顯示關卡進度條、目前波次、每波倒數，換波與 Boss 登場會跳出提示；10 波結束後進入 Boss 戰。
- 通關會解鎖並選定下一關，結算面板會列出金幣、龍晶、擊破數、吞彈、反吐、受擊、大絕、共鳴與戰鬥評價；失敗也會保留部分資源並給出下一場操作建議。
- 養成分頁的寶物裝備以六角晶體卡片排列，點擊晶體或強化按鈕可消耗金幣提高裝備等級，生命、攻擊、速度、吞噬、防禦與大絕會分別套用；技能搭配可選 1 到 3 張戰術卡，技能研究等級越高槽位越多；掛機收益會顯示可領取金幣並可一鍵領取。
- 鍵盤也可用方向鍵或 WASD 移動，按住空白鍵吸收、放開空白鍵反吐。

- 砂遺古龍 / 砂遺秘卷：沙漠遺跡，古砂秘寶。
- 櫻風舞龍 / 櫻風花鈴：櫻花輔助，櫻風花燈。
- 烈焰椒龍 / 辣焰椒冠：辛辣爆發，辣焰飛椒。
- 星夢幻龍 / 星夢法環：星空幻術，星夢回流。
- 泡泡海龍 / 泡海珍珠：泡泡治癒，泡海漩渦。
- 森語芽龍 / 森語芽杖：森林守護，森語再生。
- 霆電迅龍 / 霆電迅核：迅捷雷擊，霆電連鎖。
- 霜晶雪龍 / 霜晶雪杖：冰霜控制，霜晶結界。
- 月影幽龍 / 月影幽刃：暗影連擊，月影逆鳴。
- 齒輪機龍 / 齒輪策盤：科技策略，齒輪震波。
